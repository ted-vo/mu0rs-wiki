# Lệnh

```
/mhp x y(có hoặc ko)
```

```
/mmana x y(có hoặc ko)
```

```
/msd x
```

Với 
- x: số lượng cần mua
- y: 1 => để mua bình nhỏ. Không cần điền auto mua bình lớn

# Yêu cầu

- ZEN

# Cách sử dụng

Khi train ở những Map hỗ trợ. Gõ lệnh tương ứng để mua.
