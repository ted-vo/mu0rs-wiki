# Lệnh

```
/afk
```

```
/afk0
```

```
/afk1
```

# Yêu cầu

| Lệnh | Phí | Mô Tả |
|:-----|:----|:------|
| `afk` | Có | -  15p / 1 coin. |
| `afk0` | Không | - Không gây sát thương.<br>- Không drop<br>- Phù hợp cho các char buff. |
| `afk1` | Có | - 15p / 2 wcoins<br>- x2 drop. |

# Cách sử dụng

Vào Game bật Auto => Gõ lệnh => Tắt máy đi chơi
